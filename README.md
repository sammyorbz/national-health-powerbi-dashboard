## Overview
This project is an interactive Power BI dashboard designed to analyze national healthcare data, focusing on patient demographics, hospital performance, treatment outcomes, and healthcare costs.

Users can explore insights dynamically by year, hospital, and medical condition (e.g. Asthma).

---

## Key Features
- Admissions, billing, doctors, bedspace, and length of stay KPIs
- Year-over-year performance comparisons
- Patient demographics by age, gender, condition, and blood group
- Hospital-level and condition-specific insights
- Treatment and cost analysis by provider and insurance

---

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling
- Bookmarks & Parameters

3. Update data folder parameter if prompted
4. Use slicers and navigators to explore insights
